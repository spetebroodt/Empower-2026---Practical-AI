# Empower 2026 - Practical AI Workflows

This package is the accompanying package for the **Showcase of practical AI workflows** session in the advanced track of  Empower 2026.
During this session, we will get handson with *Relational Anomaly Detection* and *Document Intelligence*.

The package deploys five elements under the view *DataMiner Catalog* > *Empower 2026* > *Relational Anomaly Detection Demo*.
You can use the elements to follow along with the exercises during the session and to explore the capabilities of relational anomaly detection.

![Elements in DataMiner Cube showcasing Relational Anomaly Detection](./Images/Screenshot.png)

The package will also deploy a few low code apps on your system:
- The Document Intelligence app, which will be needed to follow along with the exercises on document intelligence.
- The RAD Manager app, which will be needed to follow along with the exercises on relational anomaly detection.

![Low code apps in the package](./Images/appMain.png)

The elements deployed by this package will read in history data slowly and should therefore be deployed during the very first
session entitled **Getting your data in seconds**. This session takes place
on Tuesday 17/03/2026 at 9h30.
