# Empower 2026 - Practical AI Workflows

## About

This package is the accompanying package for the **Showcase of practical AI workflows** session in the advanced track of Empower 2026.

During this session, we will get hands-on with *Relational Anomaly Detection* and *Document Intelligence*.

### Elements

The package will deploy five elements under the *DataMiner Catalog* > *Empower 2026* > *Relational Anomaly Detection Demo* view.

You can use these elements to follow along with the exercises during the session and to explore the capabilities of relational anomaly detection.

The elements will read in history data slowly, and should therefore be deployed during the very first session named **Getting your data in seconds**.

![Elements in DataMiner Cube showcasing Relational Anomaly Detection](./Images/Screenshot.png)

### Low-code applications

The package will also deploy the following low-code apps on your system:

- The **Document Intelligence** app, which will be needed to follow along with the exercises on document intelligence.
- The **RAD Manager** app, which will be needed to follow along with the exercises on relational anomaly detection.
